# GoLive App 🎥

A full-stack live streaming app using RTMP, Socket.IO, FFmpeg, React (Vite), and Node.js with Docker.

## 📦 Tech Stack

- Frontend: React + Vite
- Backend: Node.js + Express + Socket.IO + FFmpeg
- Streaming: RTMP (YouTube, Twitch, etc.)
- Docker + Docker Compose

## 🚀 Running the App

```bash
git clone https://github.com/yourusername/go-live-app.git
cd go-live-app
docker-compose up --build
